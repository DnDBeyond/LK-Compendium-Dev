FRAIF COMPONENTS

HEADERS
- h1 shells the chapter masthead with --fraif-h1-text-color pulling from the relevant regional gradient and a -webkit-text-stroke outline (fraif.css:200). Override --fraif-h1-text-color on the heading element to swap gradients.
- h2 and h3 default to Wolpe Pegasus small-caps with --header-underline available for underline-filigree pairings (fraif.css:214).

DROP CAPS
- .fraif-dropcap sets --fraif-dropcap-art and --fraif-dropcap-fill. Regional overrides are applied per chapter via ID selectors such as #AboutTheDalelands and #AboutIcewindDale (fraif.css:760+).
- Apply the class to the first paragraph in a sidebar or lead block:
```
<p class="fraif-dropcap">
  The Dalelands are broad forest vales with rolling farmlands...
</p>
```

GAZETTEER QUOTE
- .gazetteer-quote renders centered quotations with inline attribution art (fraif.css:423).
- Use data-by on the final paragraph; position the attribution by setting CSS custom properties on that element.
```
<aside class="gazetteer-quote">
  <p>"The first time you walk into a Dalelands tavern..."</p>
  <p data-by="Erevan of Featherdale"
     style="--y:62; --y-mobile:68; --x-left:54; --x-right:32;"> </p>
</aside>
```

GAZETTEER SIDEBAR
- Combine .aside--nine-slice with .gazetteer-sidebar for chapter sidebars (fraif.css:827).
- --gazetteer-sidebar-accent controls the gradient and drop-cap colors; prebuilt IDs (#AboutTheDalelands, #AboutIcewindDale, etc.) set the correct asset pairings and CDN art. For custom sidebars, override:
```
<aside class="aside--nine-slice gazetteer-sidebar"
       style="--gazetteer-sidebar-accent: var(--calimshan-secondary);
              --ns-image: url('https://media-stg.dndbeyond.com/compendium-images/fraif/ui/fraif-gazetteer-border-green.png');">
  <p class="fraif-dropcap">Calimshan bustles with trade...</p>
</aside>
```

LOCATION BANNER
- h3.location-banner wraps location headers in a masked polygon with twin filigree images (fraif.css:588).

<h3 class="compendium-hr location-banner"
    id="MythDrannor"
    style="--location-banner-shape-color: var(--dalelands-primary);
           --location-banner-secondary-color: var(--dalelands-secondary);
